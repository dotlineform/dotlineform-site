# Dreaming is Active: Flow of Memory Reconstruction

Dreaming participates in a recurrent loop that reactivates, edits, and integrates memory traces — especially across NREM (systems consolidation) and REM (emotional recalibration) phases.

### How to read this

- **Replay** during NREM/REM reactivates traces and biases dream content.
- **Construction** in REM is generative (activation–synthesis / predictive processing).
- **Reconstruction** performs *memory triage*: selection, generalization, weakening.
- **Emotional recalibration** reduces negative affect and preserves informational content.
- Outputs are **integrated** into cortical networks, shaping next‑day behavior and future recall.

## Why this is an *active* loop

1. **Replay influences content**: Dream imagery tracks pre‑sleep experience and hippocampal reactivation.
2. **Selection & abstraction**: Sleep performs discriminatory consolidation (“triage”) that edits memory.
3. **Affect remodulation**: REM down‑tunes amygdala reactivity and supports emotional homeostasis.
4. **Model updating**: Predictive processing treats dreams as offline simulations that refine generative models.

Together these mechanisms show dreaming is not passive background noise but an *active reconstruction engine*.
