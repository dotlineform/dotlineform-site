# exploreCurve


 <span style="background-color: #00C6BD;">
     how would i use exploreCurve function to create an interactive sketch where I can dynamically set the vector ink options?
 </span>

You can make your sketch interactive by providing GUI controls (for example, sliders) that update the parameters for your vector ink options, and then use these updated values every frame to re‑draw your curve via the exploreCurve function. One common way in Processing is to use the ControlP5 library. For example:
	1.	**Import and initialize ControlP5 in setup():**
Create sliders for parameters such as distortionScale, noiseScale, inkFraction, exponent, slope, initialMultiplier, velocity, and numParticles.
	2.	**In draw():**
Read the current slider values (they will update global variables automatically if you name the sliders the same as your variables), then create a new VectorInkOptions object with those values. Call exploreCurve (or an updated version that uses these vectorInkOptions) so that the curve is redrawn using the current slider values.

**How It Works**
	•	**ControlP5 Sliders:**
The sliders are created in setup() and linked to global variables. When a slider is moved, the global variable (like distortionScale) is updated automatically.
	•	**Dynamic VectorInkOptions:**
In draw(), the current slider values are used to create a new VectorInkOptions object that is passed to exploreCurve. This means that every time draw() is called (or if you call redraw()), the curve will be generated with the latest parameters.
	•	**Interactive Redrawing:**
If you want the sketch to update continuously as you adjust the sliders, remove noLoop() so that draw() is called every frame. Alternatively, you could call redraw() in the ControlP5 callback.
	•	**Passing Options:**
The updated exploreCurve now accepts a VectorInkOptions object. It uses these values when calling drawCurve_VectorField to create the curve interactively.

This approach gives you a dynamic, interactive sketch where you can adjust vector ink options on the fly, and the curve will update accordingly.
