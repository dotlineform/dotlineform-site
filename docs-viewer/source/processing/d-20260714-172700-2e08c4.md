# Controlling the Degree of Self-Mutation


This guide provides techniques for managing how much and how often variables mutate themselves in static generative drawings — particularly in Processing. Even without animation, self-mutating variables can be used to generate complex, unrepeatable visual outcomes. These techniques help control the degree of variation and complexity at the moment of generation.

This guide applies to static generative compositions and complements the [[Self-Mutating Variables]] and [[When Variables Become Functions]].

 <span style="font-family: .AppleColorEmojiUI;">
     ## 🎚
 </span> **1. Mutation Scaling Factor**


```
float angle = 0;
float mutationScale = 0.005;

void setup() {
  angle += sin(angle) * mutationScale;
  // Use angle to define points, shapes, etc.
}

```



 <span style="font-family: .AppleColorEmojiUI;">
     ## 🕳
 </span> **2. Damping and Decay**


```
float velocity = 0;
float value = 0;
float damping = 0.95;

void setup() {
  for (int i = 0; i < 100; i++) {
    velocity += sin(value) * 0.01;
    velocity *= damping;
    value += velocity;
    // Use value to influence drawing
  }
}

```

 <span style="font-family: .AppleColorEmojiUI;">
     ## 

 </span> <span style="font-family: .AppleColorEmojiUI;">
     ## 🎛
 </span> **3. Smoothstep or Sigmoid Bounds**


```
float input = random(1);
float mutation = input * input * (3 - 2 * input); // smoothstep

```

 <span style="font-family: .AppleColorEmojiUI;">
     ## 

 </span> <span style="font-family: .AppleColorEmojiUI;">
     ## 🔀
 </span> **4. Intermittent Mutation**


```
for (int i = 0; i < 500; i++) {
  if (i % 10 == 0) {
    offset += random(-0.1, 0.1);
  }
  // Use offset to determine vertex or control point
}

```

 <span style="font-family: .AppleColorEmojiUI;">
     ## 

 </span> <span style="font-family: .AppleColorEmojiUI;">
     ## 🧬
 </span> **5. Meta-Variable Controllers**


```
float rate = random(0.05);
float evolve(float val) {
  return val + sin(val) * rate;
}

```

 <span style="font-family: .AppleColorEmojiUI;">
     ## 

 </span> <span style="font-family: .AppleColorEmojiUI;">
     ## 🧠
 </span> **6. Feedback-Constrained Mutation**


```
if (abs(x) > width / 2) {
  mutationScale *= -1;
}

```

     ## 🛠

```
float mutationScale = map(100, 0, 255, 0, 0.05); // simulate user tuning

```
