# line parameters


So createLine takes the following parameters:

styleOptions
curveOptions 
boundaryOptions

actually, boundaryOptions can be merged into curveOptions

lineStyle 
lineBoundary


## simplify the parameters

**lineBoundary**

this defines the line placement and the area which it can cover


**lineStyle**

this defines what the line line looks like
