# drawLine demo


Below is an implementation for a new function, **demo_drawLine()**, which draws a line for each style (“quick”, “natural”, “wild”) in a manner similar to **demo_createLineBounded()**. In this example, we use the existing **drawLine()** function and simply loop through the styles while spacing the start points vertically:

void demo_drawLine() {
  *// Define the styles you want to demonstrate.*
  String[] styles = {"quick", "natural", "wild"};
  
  *// Set a fixed starting x-coordinate.*
  float startX = 200;
  
  *// Calculate vertical spacing based on the number of styles.*
  int numStyles = styles.length;
  float ySpacing = height / (numStyles + 1);
  
  *// Set text properties for labeling.*
  textAlign(RIGHT, CENTER);
  textSize(12);
  fill(0);
  
  *// Loop through each style, draw the line, and label it.*
  for (int i = 0; i < numStyles; i++) {
    *// Calculate the start point for this style.*
    PVector start = new PVector(startX, ySpacing * (i + 1));
    
    *// Draw the line using the specified style.*
    drawLine(start, styles[i]);
    
    *// Optionally, add a label (the drawLine function already labels the line).*
    *// If you want an additional label, you could do:*
    *// text("Style: " + styles[i], start.x - 20, start.y);*
  }
}

**How It Works**
	•	**Styles Array:**
The function defines an array of style strings—“quick”, “natural”, and “wild”.
	•	**Spacing:**
It uses a fixed starting x-coordinate and evenly spaces the y-coordinates across the height of the canvas.
	•	**Loop:**
For each style, it creates a start point, calls **drawLine()** (which uses **setLineBoundary()** internally with the given style), and labels the line.

Simply call **demo_drawLine()** (for example, in your setup or key press handler) to see a demonstration of the three different styles in one drawing.
