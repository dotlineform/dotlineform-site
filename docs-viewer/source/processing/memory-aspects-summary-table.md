*Scope:* This summary is derived from the uploaded article “Memory Stages: Encoding, Storage and Retrieval” by Saul McLeod (June 16, 2023) on SimplyPsychology.

## Summary Table

| Aspect | Definition / Key Idea | Notes & Examples |
| --- | --- | --- |
| **Encoding** | Transforming incoming information into a form the memory system can store (visual, acoustic, semantic). | Seeing a number (visual), repeating it aloud (acoustic), or remembering meaning (semantic). |
| **Storage** | Maintaining information across time, including capacity and duration of different stores. | STM ≈ 5–9 items for ~0–30s (Miller’s “7 ± 2”); LTM ~ unlimited capacity and potentially lifelong duration. |
| **Retrieval** | Accessing stored information; mechanisms differ by store. | STM retrieval is often sequential; LTM retrieval is associative and cue/context dependent. |
| **Rehearsal** | Verbal/mental repetition that strengthens encoding and supports transfer from STM to LTM. | Keeping items active by repeating them; effective regardless of visual or auditory presentation. |
| **Ecological Validity** | How well memory findings (often from lab tasks) generalize to real-life settings. | List-learning tasks may be artificial; real-world applicability can vary. |
