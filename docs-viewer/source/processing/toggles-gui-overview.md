# On‑Canvas Toggles & Sliders

> Timestamp: 2025-08-29 18:59:14 BST

This build adds live controls to the existing folder‑picker GUI:

- Checkboxes: **UHD MODE**, **UHD STRICT**, **FORCE GREYSCALE**, **AUTO LEVELS**, **PER‑CHANNEL** (Auto Levels)
- Cycle button: **BLEND STRATEGY** (`SEQUENTIAL`/`AVERAGE`)
- Sliders: **LAYER OPACITY** (0–1), **CLIP PERCENT** (0–5%)

Everything else remains: CSV per‑layer rules, manifest logging, Auto Levels/Contrast with percentile clipping, permutations generator, and two blend strategies.

If you’d like a progress bar, keyboard shortcuts, or a “dry‑run” preview of the first N permutations, say the word.
