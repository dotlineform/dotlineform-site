# InkEngine

InkEngine is the recovered Processing sketch formerly named
`sketch_250309a`. Its source is in:

```text
processing/projects/InkEngine/
```

The sketch compares vector-field, particle, diffuse-particle, and
textured-particle interpretations of the same curve, then overlays them as a
weighted composite. ControlP5 supplies the authoring controls.

The recovered baseline builds and runs with Processing 4.5.5 in Java mode and
ControlP5 2.2.6. Named slider configurations are stored in the tracked
`data/sliderSettings.json` file so useful parameter combinations can be
recalled.

The existing child documents are historical notes, code experiments, and
partial explanations created while the sketch evolved. They are evidence for
reconstructing the design, not a current specification.
