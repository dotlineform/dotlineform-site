# When Variables Become Functions


this complements [[Self-Mutating Variables]]

This guide explores a powerful evolution in generative art: transforming static variables into dynamic functions. This transformation allows for emergent, time-sensitive, and reactive behavior in creative code, particularly in Processing.

 <span style="font-family: .AppleColorEmojiUI;">
     ## 

 </span> <span style="font-family: .AppleColorEmojiUI;">
     ## 🔁
 </span> **Why Turn a Variable into a Function?**


```
float speed = 2.0;

```


**becomes:**


```
float speed() {
  return sin(frameCount * 0.01) * 5;
}

```


* The value updates over time
* The logic becomes portable and composable
* The system exhibits feedback-like behavior

## 🎛 **Basic Example: Dynamic Radius**


```
float radius() {
  return 100 + sin(frameCount * 0.05) * 20;
}

ellipse(width/2, height/2, radius(), radius());

```


The radius now oscillates over time.

## 🧠 **With Memory (State Retention)**


```
float base = 100;
float offset = 0;

float radius() {
  offset += noise(frameCount * 0.01) * 0.05;
  return base + sin(offset) * 40;
}

```


This function accumulates state, evolving non-linearly.
