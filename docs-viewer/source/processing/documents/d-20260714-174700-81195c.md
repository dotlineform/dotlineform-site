# UI positioning


<instructions>
- In the setup() function, find the line that defines "int sliderX = 250;" and change it to "int sliderX = 1000;".
- In the setup() function, update the positions of the refresh and save buttons: change their x positions from 40 to 1020.
- In generateSettingButtons(), change the line "int baseX = 250;" to "int baseX = 20;".
- In demo_drawCurve_Freedom(), change the declaration "int startOffsetx = 200;" to "int startOffsetx = 1000;".
</instructions>
