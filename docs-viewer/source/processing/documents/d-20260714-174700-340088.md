# drawCurve (PVector start, int freedom, StrokeStyle strokeStyle)

## drawCurve (PVector start, int freedom, StrokeStyle strokeStyle)


draws a Bezier curve, geometry derived from the parameters.
returns a BezierCurve object variable containing the control points etc

### **parameters**

 <span style="background-color: #00C6BD;">
     **start**
 </span> - the start point of the curve

 <span style="background-color: #00C6BD;">
     **freedom**
 </span> **-** how the curve is constrained to a boundary, defined in [[defineCurve (PVector start, int freedom, int direction)]]

 <span style="background-color: #00C6BD;">
     **strokeStyle**
 </span> - what the curve looks like:

strokeStyle.media:
- ink
- pencil
strokeStyle.weight: 
- light
- medium
- bold

The drawCurve function implements strokeStyle as follows:

strokeStyle.media maps to which drawCurve_ function(s) should be used to draw the curve, and what inkOptions to use.
strokeStyle.weight maps to lineOptions.lineWidth (and potentially inkOptions if a mapping can be defined)
