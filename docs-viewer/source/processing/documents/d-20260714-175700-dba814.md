# Checklist — From Layering to Optical‑Flow Interpolation

Prepared: 2025-09-02 16:12:52 (Europe/London). Tick items as you complete them. This page stores state only while open (no persistence).

M0

— Set up baseline: load A/B and layer with fixed alpha; verify output folder & save one frame.

M1

— Animate true crossfade with easing; export sequence.

M2

— Switch to gamma‑correct crossfade (GLSL shader).

M3

— Normalize inputs to fixed canvas (FIT/FILL/STRETCH) and verify sizing.

M4

— Optional: Add homography alignment pre‑step for planar scenes.

M5

— Compute multi‑scale Horn–Schunck dense flow (A→B).

M6

— Implement bilinear warping of A and B using flow; premult + gamma‑correct blending.

M7

— Add per‑pair flow caching (binary); test resume behaviour.

M8

— GPU warper: pack flow to RG texture + shader decode; disable mipmaps.

M9a

— Quality: forward–backward consistency mask for occlusions.

M9b

— Quality: small‑hole fill (edge‑aware or median/bilateral).

M9c

— Quality: multi‑band (Laplacian pyramid) blend for seams.

M9d

— Quality: time easing curve options (smoothstep/cubic/custom).

M10

— Batch over large image set; generate sequential PNGs; stitch with FFmpeg.
