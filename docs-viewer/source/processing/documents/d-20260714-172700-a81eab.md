# Entropy-Mapped Bezier Fields


Entropy-Driven Vector Fields for Generative Bézier Curves

Design a system where Bézier ink curves are influenced by dynamic entropy sources — including randomised vector fields, noise patterns (e.g. Perlin, Simplex), and real-time entropy spikes. These fields govern the control points of each curve, allowing strokes to emerge unpredictably and reactively, simulating organic or quantum-like behaviours across the canvas.

Includes:

* Random vector field generation
* Entropy-sensitive control point modulation
* Reactive deformation to local entropy gradients
* Optional real-time mutation during rendering

**Suggested Follow-up Prompt**
Simulate a Bézier engine where curves evolve under noise- and entropy-driven fields. What happens if entropy spikes cause reactive control point changes mid-stroke?
