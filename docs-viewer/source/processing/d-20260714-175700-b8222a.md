# Progress Bar & “First N” Limits

> Timestamp: 2025-08-29 19:12:38 BST

## What’s new

- **Live progress bar** while generating composites. The UI stays responsive.
- **First N** controls for *Preview* and *Final* so you can cap the number of permutations processed.

## How it works

The generator now runs in a separate thread, so the draw loop can update the progress bar and counters. The bar shows how many composites have been written out of either the full permutation count or the selected *First N* cap.

## Tips

- Set a small *First N (Preview)* (e.g., 50–200) when exploring settings.
- Use *Preview Scale* (10–50%) for rapid iteration, then set *First N (Final)* to 0 ( = all ) for the full output.

If you want a *Cancel* / *Pause* button, we can add that next by checking a flag inside the worker loop.
