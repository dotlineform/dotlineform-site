# Introduction



```
Composites: how many unique number sequences can you make from numbers 1,2,3?

```

Let’s slow this down and be careful — because your question has some ambiguity.

**“Composites: how many unique number sequences can you make from numbers 1,2,3?”**

This can mean a few different things, so I’ll unpack each possibility:

⸻

**1. If you mean “all possible finite sequences made using digits {1,2,3}”**
	•	If there’s **no limit on sequence length**, the number is **infinite** (e.g., 1, 2, 3, 11, 12, 13, … forever).

⸻

**2. If you mean “how many sequences of length n you can form”**
	•	At each position you have 3 choices (1, 2, or 3).
	•	So the formula is:
\text{Total sequences of length n} = 3^n
	•	Examples:
	•	Length 1 → 3^1 = 3 sequences: {1, 2, 3}
	•	Length 2 → 3^2 = 9 sequences: {11, 12, 13, 21, 22, 23, 31, 32, 33}
	•	Length 3 → 3^3 = 27 sequences, etc.

⸻

**3. If you mean “unique sequences using each of 1, 2, 3 exactly once” (permutations)**
	•	Then it’s just the permutations of 3 numbers:
3! = 6
	•	Sequences: {123, 132, 213, 231, 312, 321}

⸻

 <span style="font-family: .AppleColorEmojiUI;">
     ✅
 </span> So the answer depends on your intention:
	•	**Infinite** (if any length allowed).
	•	3^n (if fixed length n).
	•	**6** (if each number must appear once).

⸻

Do you want me to compute the **total number of unique sequences of all lengths up to 3**, using numbers 1,2,3? That seems like it might be what you’re asking by “composites”.
