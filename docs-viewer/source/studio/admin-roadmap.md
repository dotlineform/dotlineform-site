# Admin Roadmap

Admin roadmap items improve repeatable evidence and operational review without turning Admin into a universal dashboard.

## Features

- [Responsibility Reports](/docs/?scope=studio&doc=admin-checks-responsibility-reports) — focused ownership and dependency evidence.
- [Operational Reports](/docs/?scope=studio&doc=admin-checks-operational-reports) — runtime, payload, generated-artifact, and performance evidence.

Each report delivery should answer one narrow question and retain its own structured evidence. Compound reports can join proven artifacts later.
