# Repository Roadmap

Repository roadmap items cross app ownership or improve the way the repository is maintained.

## Features

- [Repository Maintenance](/docs/?scope=studio&doc=repository-maintenance) — curated links to concrete gaps retained in durable documentation.
- [Search Mechanics](/docs/?scope=studio&doc=repository-search-mechanics) — index composition, ranking, and efficiency inside the current search structure.
- [Source Documentation And Linting](/docs/?scope=studio&doc=repository-source-documentation-and-linting) — possible JSDoc, docstring, Ruff, and ESLint conventions.

The maintenance index is not a backlog. The other features remain ideas until a coherent delivery is worth scheduling.
