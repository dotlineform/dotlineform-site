# Docs Viewer Source Config Report

This manage-mode report shows the Docs Viewer source config, browser projection, and generated artifact projection for every configured scope.

The generated projection reads viewer options from each scope's configured `index-tree.json` output, not from the retired flat docs `index.json`.
