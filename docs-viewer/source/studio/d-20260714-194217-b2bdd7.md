# External Asset Collections

## Purpose

Give local Docs Viewer scopes a useful inventory of user-owned project files without turning every file into a Markdown document or making Docs Viewer own the files.

This is the feature parent. It keeps the broad idea, architecture, and implementation-ready deliveries separate.
