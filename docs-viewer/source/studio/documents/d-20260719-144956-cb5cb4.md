# New Doc

## public scopes

**source**

documents: `docs-viewer/source/analysis/documents`  
media: `docs-viewer/source/analysis/media`  

- `docs-viewer/` -> `source/` -> `<scope>` -> content

**published**

documents: `site/assets/data/docs/scopes/analysis/by-id`  
media: `site/assets/data/docs/scopes/analysis/media`  

- `site/assets/data/` -> `docs/` -> `scopes/` -> `<scope>` -> content

## local scopes (in-repo)

## external local scopes

**source**

documents: `docs-viewer/source/notes/documents`  
media: `docs-viewer/source/notes/media`  

- `docs-viewer/` -> `source/` -> `<scope>` -> content

**published**

documents: `docs-viewer/published/docs/notes/by-id`  
media (built): `docs-viewer/source/notes/media`  
media (finished): `docs-viewer/source/notes/media`  

- `docs-viewer/` -> `published/` -> `docs/` -> `<scope>` -> content

- *published* media is saved under `source/...`
- *imported* svg and *built* svg are saved in same folder `docs-viewer/source/notes/media/svg` (so file collisions are possible)
