# Docs Viewer Roadmap

## Current Focus

Diagrams is shipped. The remaining pre-transition sequence completes the bounded External Asset Collections association/inventory outcome and Local Tree Move Projection, then takes Word Document Import before the Analytics ownership transition and Docs Viewer Core Refactor.

Bug fixes, regressions, safety work, and small maintenance continue normally. Paused features remain visible after Standalone Docs Viewer for later reconsideration.
