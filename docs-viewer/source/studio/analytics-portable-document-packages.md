# Portable Document Packages

Explore Data Sharing packages that preserve canonical document source and its owned dependencies when rendered-content exchange is insufficient.

## Feature Documents

- [Full Document Export Delivery](/docs/?scope=studio&doc=data-sharing-full-document-package-delivery) — current source-faithful, export-only package work.

Current durable Data Sharing architecture remains authoritative. This feature does not imply that source-faithful packages support return import; any future return path needs its own reviewed validation and apply boundary.
