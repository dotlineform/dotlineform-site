## Retirement Position

Analytics no longer hosts document-package or tag-package workflows. Integrated Docs Viewer owns [Share Document Packages](/docs/?scope=studio&doc=d-20260718-155350-c84e62); tag-package sharing was retired without replacement. The remaining Analytics tag routes stay in place until [Tag Management In Studio](/docs/?scope=studio&doc=d-20260718-155350-a2d49c) moves their canonical data and write paths.
