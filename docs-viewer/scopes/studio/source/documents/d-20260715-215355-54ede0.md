# Local Tree Move Projection

## Purpose

Replace the routine full-index refresh after a successful single-document move with a local projection of the server's committed result.

This is a bounded, independent refactor with an immediate benefit: current drag/drop preserves the displayed document, unrelated nodes, expansion state, and other local context.

## Status

The feature is active. The committed-record model and mounted-tree projection, targeted cache invalidation, and explicit recovery branch are implemented with focused evidence. Route-level move evidence and durable runtime closeout remain before the feature is marked shipped. It does not add checkbox selection or group movement.
