# Portable Document Packages

Explore a later integrated Docs Viewer package that preserves canonical document source and its owned dependencies when rendered-content exchange is insufficient.

## Feature Documents

- [Full Document Export Delivery](/docs/?scope=studio&doc=d-20260711-012238-8a04a6) — proposed source-faithful, export-only package work.

The shipped [Share Document Packages](/docs/?scope=studio&doc=d-20260718-155350-c84e62) boundary remains authoritative. No `document-full-source` profile is enabled. This feature does not imply that source-faithful packages support return import; any future return path needs its own reviewed validation and apply boundary.
