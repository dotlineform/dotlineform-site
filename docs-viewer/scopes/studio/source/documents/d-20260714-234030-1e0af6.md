# Docs Viewer Roadmap

## Current Focus

Diagrams, Word Document Import, and Scope-First Storage Layout are shipped. The remaining pre-transition work is Scope Media Management inventory, Local Tree Move Projection, and Document Packages In Docs Viewer. Docs Viewer Core Refactor follows Document Packages and Analytics App Retirement; Standalone Docs Viewer remains its first independent consumer.

Bug fixes, regressions, safety work, and small maintenance continue normally. Paused features remain visible after Standalone Docs Viewer for later reconsideration.
