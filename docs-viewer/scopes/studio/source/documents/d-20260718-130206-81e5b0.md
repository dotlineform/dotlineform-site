# Search Roadmap

Search remains two domain-owned static-index systems: Catalogue Search and Docs Viewer Search. This roadmap coordinates features that need deliberate ordering across those domains without creating a shared search application, runtime, or repository-maintenance owner.

| suggested order | feature | status | depends on | outcome |
| ---: | --- | --- | --- | --- |
| 1 | [Search Mechanics](/docs/?scope=studio&doc=d-20260715-092247-6bbb5e) | paused | — | review inclusion, ranking, context, and efficiency, then identify independently finishable improvements while retaining the separate search systems |

Add another feature only when it has a coherent search outcome distinct from the mechanics review. Keep Catalogue-specific and Docs Viewer-specific implementation under their domain owners.
