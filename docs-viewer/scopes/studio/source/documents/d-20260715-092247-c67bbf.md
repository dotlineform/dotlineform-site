# Analytics Roadmap

Analytics roadmap items cover semantic models, tag workflows, and Data Sharing capabilities owned by Analytics.

## Features

- [LLM Integration](/docs/?scope=studio&doc=d-20260416-000000-1744c6) — advisory semantic assistance with domain-owned review and apply boundaries.
- [Tag Dimensions](/docs/?scope=studio&doc=d-20260715-092247-659e29) — exploratory dimension definitions, representations, scoring, and projections.
- [Portable Document Packages](/docs/?scope=studio&doc=d-20260715-092247-4b7382) — source-faithful Data Sharing packages, currently limited to export.
- [Shareable Content Blocks](/docs/?scope=studio&doc=d-20260623-000000-8b3359) — bounded source exchange and reviewed writeback.

## Current Position

The portable full-document export is the only in-progress delivery. LLM Integration and Tag Dimensions remain paused ideas. Shareable Content Blocks remains conceptual.
