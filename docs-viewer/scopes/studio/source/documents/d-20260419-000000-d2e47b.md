# Dev Home

## Repository Shape

- `site/` is the checked public GitHub Pages artifact. There is no deploy-time copy/build step.
- [Studio](/docs/?scope=studio&doc=d-20260423-000000-d015e6) edits catalogue/editorial source and builds public catalogue payloads.
- [Analytics](/docs/?scope=studio&doc=d-20260506-181900-d08dbe) owns canonical tag workflows and hosts Data Sharing.
- [Docs Viewer](/docs/?scope=studio&doc=d-20260424-000000-50b63f) renders/manages document scopes; public routes reuse site-owned assets/config.
- [Admin](/docs/?scope=studio&doc=d-20260606-212412-01ed44) runs/reviews operational audits, checks, activity, and test summaries.
- [Processing](/docs/?scope=studio&doc=d-20260720-101826-994dd0) owns repository hosting and integration for the separate Java/Processing project; its own Docs Viewer scope retains conceptual and technical documentation.
