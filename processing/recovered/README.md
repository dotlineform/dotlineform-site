# Recovered Processing Source

This directory preserves related source snapshots that are not yet maintained
as runnable Processing projects.

The filenames record the apparent evolution of each experiment. Do not merge,
rename, or discard versions solely because a later filename looks more
complete. First identify dependencies, inputs, known output, and the version
that should become the runnable baseline. Promote that version into
`processing/projects/<SketchName>/<SketchName>.pde` only after it builds.
