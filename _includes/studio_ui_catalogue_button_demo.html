Default Command Button

Current shared Studio baseline for command actions.

Use for direct commands such as save, open, import, or create.

Anchor Command

Shared button chrome applied to navigation that behaves like a command action.

Used on current Studio editor pages for actions such as creating a new related record.

New Detail

Disabled

Current disabled treatment before richer emphasis variants are defined.

The shared button remains the same size and geometry when disabled.

Modal Action Row

Buttons inside the shared modal action container.

Modal buttons are in scope for this primitive. The row layout belongs to the modal shell; the button shape stays shared.

Toolbar Subset

Command buttons used inside a toolbar or action strip.

Toolbar buttons are a subset of command buttons. Future icon-only toolbar actions should still inherit the same shared geometry.
