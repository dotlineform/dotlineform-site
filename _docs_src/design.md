# Design

This section covers the current visual and interaction standards for the site.

Use this section for:

- site-wide UI rules and interaction defaults
- docs-viewer and public-search UI standards
- Studio shared UI patterns
- shared CSS primitives and CSS cleanup guidance

This section does not try to describe:

- runtime data flow
- generated artifact ownership
- detailed payload or worksheet schemas
